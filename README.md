# EY Tools

Open Rails console in Engine Yard.

## Installation

Add this line to your application's Gemfile:

    gem 'ey_tools'

And then execute:

~~~ sh
$ bundle
~~~

## Usage

~~~ sh
$ ey-console
~~~
